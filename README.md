## About

**Daily 5 Questions** is a static HTML quiz website designed for revision practice.

Users:
- answer **5 written questions**
- click **Mark My Answers**
- receive instant feedback:
  - **Correct**
  - **Partly correct**
  - **Not enough yet**

The app runs entirely in the browser and does **not** require a backend or database.

---

## Features

- Clean single-page revision interface
- Daily-style 5-question practice sets
- Written-answer only format
- Instant in-browser marking
- Feedback based on:
  - key ideas
  - important terms
  - scripture references
  - criticism points
- Progress bar and overall score
- Topic-based question bank
- Responsive layout for desktop and mobile
- Works as a static site on GitHub Pages

---

## Tech stack

This project is built with:

- **HTML**
- **CSS**
- **Vanilla JavaScript**

No frameworks, build tools, or external dependencies are required.
